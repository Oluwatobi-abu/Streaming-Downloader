MyIDM Streaming Downloader
==========================

Thank you for downloading MyIDM!

How to use:
1. Extract this ZIP anywhere on your computer.
2. Double-click MyIDM.exe to launch the downloader.
3. Paste a streaming page URL (YouTube, etc.) and click "Download Stream".
4. The video will be saved in your Downloads folder.

Notes:
- FFmpeg is already bundled in ffmpeg_bin/, no need to install separately.
- Works on Windows 10/11 with no extra setup.
